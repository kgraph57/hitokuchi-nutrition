# ひとくち処方 — Concept Sheet

> **「食べたいのに、食べられない」を「おいしく、ひとくちで」解決する**

---

## Executive Summary

「ひとくち処方」は、医師が栄養設計した機能性アイスクリームブランドです。
高齢者のサルコペニア予防、がん化学療法中の栄養補給、つわり中の妊婦、偏食の子どもなど、
**「食べられない人」全員**に向けて、アイスという嗜好性の高い形態で必須栄養素を届けます。

森永グループとの共同開発により、Eルチン（特許成分）配合 × 医師監修 × 全国流通を実現し、
**年間1,000〜2,000万人の「食べられない」を解決する国民的ブランド**を目指します。

## 1. 課題：「食べられない」は巨大な未解決問題

### 対象人数（日本国内）

| セグメント | 推計人数 | 主な原因 |
|---|---|---|
| フレイル・サルコペニア高齢者 | 400〜700万人 | 食欲低下、嚥下困難、味覚変化 |
| がん化学療法中の患者 | 30〜56万人/年 | 悪心、味覚障害、口内炎 |
| つわり中の妊婦 | 25〜64万人 | 匂い過敏、食欲消失 |
| 偏食・少食の子ども | 600〜900万人 | 感覚過敏、食わず嫌い |
| 術後・入院患者 | 50〜100万人 | 術後食欲不振、嚥下制限 |
| 摂食障害 | 21〜100万人 | 食べることへの恐怖・抵抗 |
| うつ病関連の食欲不振 | 100〜140万人 | 食欲消失、調理不能 |
| **合計（重複あり）** | **1,000〜2,000万人以上** | |

### 既存の解決策とその限界

| 既存手段 | 問題点 |
|---|---|
| 経腸栄養剤・濃厚流動食 | 「薬品感」が強く、心理的に食べたくない |
| 栄養補助ドリンク（エンシュアなど） | 味が単調、飽きる、嗜好性が低い |
| サプリメント | 食事の代替にならない、満足感がない |
| 一般的なアイスクリーム | 栄養が偏っている（糖質・脂質過多） |

**ギャップ**: 「おいしくて、食べたくなって、しかも栄養が摂れる」ものがない。

## 2. ソリューション：医師が処方設計した機能性アイス

### なぜ「アイス」なのか — 医学的根拠

| 医学的メリット | エビデンス |
|---|---|
| **嚥下しやすい** | 口腔内で融解し嚥下調整食コード1j〜2-1相当に変化（学会分類2021） |
| **唾液分泌を促進** | 冷感刺激が三叉神経-迷走神経反射を介して唾液分泌を促進（Logemann 1995） |
| **化学療法中の口腔粘膜保護** | Oral Cryotherapy: 冷却が口腔粘膜炎リスクを有意に低減（Cochrane Review, RR 0.56）推奨グレードA（MASCC/ISOO 2019） |
| **匂いが少ない** | 冷温で揮発性芳香成分の放散が抑制され、つわり中でも受容される |
| **調理不要** | 開封するだけで摂取可能。うつ・術後の気力低下時に最適 |
| **水分補給にも寄与** | アイス100gあたり水分55-65g。口渇感低下の高齢者の水分摂取手段として有効 |
| **心理的ハードルが低い** | 「薬」「栄養剤」ではなく「おやつ」として摂取できる |

## 3. 栄養設計（セグメント別）

### 3-1. Silver（サルコペニア予防・高齢者向け）

**設計根拠**: PROT-AGE推奨 1.0-1.2g/kg/日、ロイシン閾値 2.5-3.0g/食

| 栄養素 | 1個あたり（100ml） | 根拠 |
|---|---|---|
| エネルギー | 200 kcal | 1日必要量の約12% |
| タンパク質 | 15g | 筋タンパク質合成に必要な1食量（Moore 2015） |
| ロイシン | 2.5g | mTORC1活性化閾値（Churchward-Venne 2012） |
| BCAA総量 | 4.5g | 骨格筋分解抑制 |
| ビタミンD | 12.5ug (500 IU) | 高齢者推奨800IUの63% |
| カルシウム | 220mg | 1日推奨量の約1/3 |
| HMB | 0.75g | ロイシン代謝産物（Flakoll 2004） |
| Eルチン | 150mg | 筋タンパク質合成促進（森永製菓特許成分） |
| 亜鉛 | 3.5mg | 筋タンパク質合成補酵素 |
| ビタミンB12 | 1.8ug | 高齢者で吸収低下 |

**タンパク質設計**: ホエイ:カゼイン = 7:3（急速+持続的アミノ酸供給）
**テクスチャー**: なめらかジェラート状。嚥下調整食コード2-1以上適合

**フレーバー**: 柑橘系・ミント系（味覚障害時に受容されやすい）

## 7. Go-to-Market 戦略

### Phase 1: 臨床実証（0-6ヶ月）

- OEM業者選定（東京堂乳業、フタバ食品等で小ロット試作）
- 勤務先病院で試食調査（n=50-100）
  - 対象: 入院高齢者（Silver）、化学療法患者（Onco）
  - 評価: 食べやすさ、味の満足度、摂取完了率
- 栄養設計の最終調整
- 森永へのアプローチ開始（プロトタイプ+初期データを持参）

### Phase 2: 製品化・初期販売（6-12ヶ月）

- 森永との協業合意、Silver（シニア向け）先行発売
- 販売チャネル:
  - クリニコ営業網で病院・介護施設
  - Amazon・楽天でD2C
  - 愛育病院（退院時配布）で妊婦向け認知
- アプリMVP開発（タンパク質トラッカー + 製品連携）
- メディア露出（医師が作ったアイスというPRストーリー）

### Phase 3: 全ライン展開 + 自治体（12-24ヶ月）

- Onco, Mama, Kids を順次発売
- 港区のフレイル予防モデル事業に共同提案（森永 x 先生）
- コンビニ・ドラッグストアへの一般流通開始
- 臨床研究データの論文化（Eルチン x 高齢者のRCT）

### Phase 4: 全国展開 + スケール（24-36ヶ月）

- 港区モデルの成功事例を横展開
- 自治体DX予算（デジタル田園都市構想交付金等）を活用
- inブランドとの正式コラボ（inひとくち処方 等）
- 海外展開調査（東南アジアの高齢化市場）
